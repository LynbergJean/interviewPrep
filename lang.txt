static vs Dynamic languages

Static: 
	-performs type checking at compile time
	-errors will show at compile time
	-declare the datatype before you use it
	-more control

Dynamic:
	-performs type checking at runtime.
	-error will not be shown till the program is ran.
	-new need to declare datatypes of variables.
	-helps save time when writing error but might give error at run time.


Strong vs weakly typed languages
		- how strict does the language enforce data types.

	Strong: 
	-you cannot mix incompatible types.
	python, java, c#

	weak:
 	-you can somtimes coerce data types.
	c, c++, javaScript
