this file will be used to add all the git commands that i know and learn about git in general:

	-mkdir 			is for creating a directory
	-cd    			is for changing directory
	-git init 		is for initializing a repository
	-git commit -m ""	is for commiting it is not up in the cloud yet up to that point
	-git push		is for pushing it to the cloud
	-touch 			is for creating a new file
	-nano 			is for editing a file 
	-vi			is also for editing a file
	-cat 			is for displaying what is in the file
	-git log		is for finding out the changes that were made to a file
	-git status 		is for getting the status of a file, is it initialized, is it committed and so forth
	-git restore --staged <>is for unadding a file that has been git add .
	-git reset <log #> 	is for resetting to a point in time, basically undoing the changes and restarting.
	-git stash		is for stashing files to an area but no pushing / commiting them
	-git stash pop		is for getting your stashed folders back
	-git stash clear	is for deleting all stashed files
	-git remote add origin  is to set up a path to the github link you will be pushing to
	-git remote -v		is to find which folder we're pusing to
	-git branch <newBranch> is to create a new branch
	-git checkout <branchName> is going to point to the branch youve made, can also git checkout main
	-git merge <branchName> is going to merge the branch changes to main
	-git clone <link>	is going to download a project into you desktop
	-git remote add upstream<url> is the upstream url from where you cloned the project from	
	-git rm path/to/file       # delete locally and from GitHub
	-git rm --cached path/to/file  # keep local, remove from GitHub
